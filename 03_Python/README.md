# Python Analysis

This folder contains all Python notebooks used for data preparation, cleaning, exploratory data analysis (EDA), and customer segmentation.

## Tools Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## Notebooks

### 01_data_loading.ipynb
- Import datasets
- Load CSV files
- Initial data inspection

### 02_data_cleaning.ipynb
- Handle missing values
- Remove duplicates
- Convert data types
- Create new columns

### 03_eda.ipynb
- Exploratory Data Analysis
- Sales trends
- Customer analysis
- Product analysis
- Seller analysis

### 04_rfm_segmentation.ipynb
- Calculate Recency, Frequency and Monetary values
- Standardize data using StandardScaler
- Perform K-Means clustering
- Create customer segments
- Export rfm_dataset.csv
