# Dataset

This folder contains the cleaned datasets used in this project.

## Files Included

- customers_clean.csv
- orders_clean.csv
- order_items_clean.csv
- order_payments_clean.csv
- order_reviews_clean.csv
- products_clean.csv
- sellers_clean.csv
- product_category_name_translation_clean.csv
- rfm_dataset.csv

## Note

The file `geolocation_clean.csv` is not included because it exceeds GitHub's 25 MB web upload limit.

The original geolocation dataset can be downloaded from the official Olist dataset on Kaggle.
